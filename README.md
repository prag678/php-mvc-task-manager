# PHP MVC Task Manager

A simple PHP MVC-based task management application demonstrating
clean architecture and CRUD operations.

# Features
- MVC architecture (Model, View, Controller)
- Task CRUD (Create, Read, Update, Delete)
- MySQL database integration
- Clean routing logic
- Simple UI

# Tech Stack
- PHP
- MySQL
- MVC Architecture
- HTML / CSS
